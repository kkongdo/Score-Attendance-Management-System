## KB국민은행 IT's Your Life 4기 1반 Mini-Project
- ✏️ 주제 : 학생과 교수가 사용하기 편리한 **성적 및 출결 관리 시스템**
- 📆 개발 기간 : 2023/08/03 - 2023/08/04
- 👨‍💻 개발 인원 : 4명
- 🔥 특이사항 : Java Console, JDBC 사용 및 MVC 패턴 적용, Oracle DB 연동

## 담당 기능
- 성적 조회, 등록, 수정 기능
- 과제 등록
